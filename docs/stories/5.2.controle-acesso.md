# Story 5.2: Controle de Acesso por Assinatura

## Status
Done

## Story
**As a** sistema,
**I want** bloquear acesso ao app para usuários sem assinatura ativa,
**so that** apenas assinantes possam usar os treinos.

## Acceptance Criteria
1. Middleware verifica assinatura ativa (status='ativa' AND fim >= NOW() - 3 days) para rotas de app
2. Sem assinatura válida → redirect para /assinatura
3. Período de graça de 3 dias após fim da assinatura antes de bloquear
4. Rotas de onboarding, auth e /assinatura acessíveis sem assinatura
5. Assinatura com status='ativa' exibe data de renovação em Configurações (futuro)
