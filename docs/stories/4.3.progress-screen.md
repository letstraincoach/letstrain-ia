# Story 4.3: Tela de Progresso

## Status
Done

## Story
**As a** usuário,
**I want** ver meu histórico, progresso de nível e conquistas em um único lugar,
**so that** acompanhe minha evolução ao longo do tempo.

## Acceptance Criteria
1. Rota /progress com nível atual + barra de progressão
2. Stats: treinos totais, streak atual, streak máximo
3. Calendário de treinos dos últimos 30 dias (dias treinados destacados)
4. Lista de conquistas (desbloqueadas em destaque, bloqueadas em cinza)
5. Link de volta para o dashboard
