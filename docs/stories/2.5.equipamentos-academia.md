# Story 2.5: Mapeamento de Equipamentos (Academia Convencional)

## Status
Done

## Story
**As a** usuário de academia convencional,
**I want** selecionar de uma lista padrão os equipamentos que tenho acesso,
**so that** o sistema saiba com o que posso treinar.

## Acceptance Criteria
1. Lista padrão de equipamentos organizados por categoria (Pesos Livres, Máquinas, Cabos, Cardio, Funcionais)
2. Busca por nome de equipamento
3. Seleção múltipla visual com check nos selecionados
4. Opção de adicionar equipamento não listado
5. Lista salva em `user_equipment`
6. Botão "Selecionar todos" por categoria
