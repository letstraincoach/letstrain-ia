# Story 4.1: Level-Up Automático

## Status
Done

## Story
**As a** usuário que completa treinos consistentemente,
**I want** subir de nível automaticamente ao atingir o número necessário de treinos,
**so that** me sinta progressivamente desafiado e motivado.

## Acceptance Criteria
1. Após salvar avaliação, sistema verifica se treinos_nivel_atual >= treinos_necessarios
2. Se sim: atualiza nivel_atual no perfil e zera contador do nível
3. Tela de level-up com animação mostrando nível anterior → novo
4. Achievement de nível desbloqueado automaticamente
5. Lógica idempotente (chamadas repetidas não causam double level-up)
