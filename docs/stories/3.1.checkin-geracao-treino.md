## Story
**As a** usuário que vai treinar,
**I want** responder 3 perguntas rápidas e receber um treino personalizado por IA,
**so that** cada treino seja adequado ao meu momento físico e mental do dia.

## Acceptance Criteria
1. Tela de check-in com 3 perguntas: (1) última refeição, (2) tempo disponível hoje (slider 20–90 min), (3) disposição de 1–10 (slider)
2. Fluxo completo de check-in realizável em menos de 30 segundos
3. Após check-in, exibir loading state com mensagem motivacional enquanto IA gera o treino
4. API route `POST /api/workout/generate` gera treino personalizado usando Claude API (claude-sonnet-4-6)
5. Treino considera: nível do usuário, local de treino, equipamentos disponíveis, objetivo, preferência, histórico dos últimos 3 treinos e check-in do dia
6. Disposição baixa (1–4): treino mais curto e leve; alta (8–10): treino na intensidade máxima do nível
7. Nenhum exercício contraindicado para lesões declaradas na anamnese
8. Treino usa APENAS equipamentos cadastrados pelo usuário
9. Treino salvo no banco com status `'gerado'` antes de exibir
10. Tempo de resposta da geração ≤ 10 segundos

## Dev Notes

### Prompt de Geração de Treino
```typescript
// src/lib/ai/workout-generator.ts
export function buildWorkoutPrompt(context: WorkoutContext): string {
  return `Você é um personal trainer especialista na metodologia Time Efficient da Lets Train.
Gere treinos rápidos (${context.tempo_disponivel} minutos), objetivos e cientificamente embasados.
Responda APENAS com JSON válido no schema especificado, sem texto adicional.

PERFIL DO USUÁRIO:
- Nível: ${context.nivel} (${LEVEL_CONFIG[context.nivel].label})
- Objetivo: ${context.objetivo}
- Preferência: ${context.preferencia_treino}
- Lesões/Restrições: ${context.lesao_descricao || 'Nenhuma'}
- Doença cardíaca: ${context.doenca_cardiaca ? 'Sim - treino leve e seguro' : 'Não'}

CONTEXTO DE HOJE:
- Local de treino: ${context.local_treino}
- Equipamentos disponíveis: ${context.equipamentos.join(', ')}
- Última refeição: ${context.ultima_refeicao}
- Tempo disponível: ${context.tempo_disponivel} minutos
- Disposição: ${context.disposicao}/10 ${context.disposicao <= 4 ? '(BAIXA - reduza intensidade e volume)' : context.disposicao >= 8 ? '(ALTA - intensidade máxima do nível)' : '(MODERADA - intensidade normal)'}

HISTÓRICO RECENTE (evite repetir os mesmos exercícios principais):
${context.historico_recente.map((t, i) => `Treino ${i+1}: ${t.exercicios_principais.join(', ')}`).join('\n')}

METODOLOGIA LETS TRAIN - TIME EFFICIENT:
- Foco em compostos multiartiulares
- Descanso ativo entre séries quando possível
- Progressão por volume e densidade, não por carga
- Para Adaptação: máximo 40% da capacidade, foco em técnica
- Para Iniciante: 60% da capacidade, aprender movimento
- Para Intermediário: 75% da capacidade, adicionar complexidade

Gere o treino em JSON seguindo exatamente este schema:
{
  "nome": "string - nome criativo do treino",
  "duracao_estimada": number,
  "aquecimento": [{ "nome": string, "series": number, "repeticoes": string, "descanso_segundos": number, "youtube_url": string }],
  "principal": [{ "nome": string, "grupo_muscular": string[], "series": number, "repeticoes": string, "descanso_segundos": number, "youtube_url": string, "instrucoes": string }],
  "cooldown": [{ "nome": string, "series": number, "repeticoes": string, "descanso_segundos": number }]
}`
}
```

### Validação da Resposta com Zod
```typescript
// src/lib/ai/workout-schemas.ts
import { z } from 'zod'

const ExerciseSchema = z.object({
  nome: z.string(),
  grupo_muscular: z.array(z.string()).optional(),
  series: z.number().int().min(1).max(10),
  repeticoes: z.string(), // "12", "8-12", "30s", "falha"
  descanso_segundos: z.number().int().min(0).max(300),
  youtube_url: z.string().url().optional(),
  instrucoes: z.string().optional(),
})

export const WorkoutSchema = z.object({
  nome: z.string(),
  duracao_estimada: z.number().int().min(10).max(120),
  aquecimento: z.array(ExerciseSchema).min(2).max(6),
  principal: z.array(ExerciseSchema).min(3).max(12),
  cooldown: z.array(ExerciseSchema).min(1).max(4),
})

export type GeneratedWorkout = z.infer<typeof WorkoutSchema>
```

### Chamada à API Anthropic
```typescript
import Anthropic from '@anthropic-ai/sdk'
const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY })

const response = await anthropic.messages.create({
  model: 'claude-sonnet-4-6',
  max_tokens: 2048,
  messages: [{ role: 'user', content: prompt }],
})
const rawContent = response.content[0].type === 'text' ? response.content[0].text : ''
const parsed = JSON.parse(rawContent)
const workout = WorkoutSchema.parse(parsed) // lança erro se inválido
```

### Tratamento de Erros na Geração
- Se Claude API falhar (timeout, rate limit): retornar erro 503 + mensagem "Tente novamente em instantes"
- Se JSON inválido: logar erro + tentar novamente 1 vez antes de falhar
- Se Zod validation falhar: logar schema inválido para debugging
