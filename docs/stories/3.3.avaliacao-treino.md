# Story 3.3: Avaliação Pós-Treino

## Status
Done

## Story
**As a** usuário que concluiu um treino,
**I want** avaliar como foi o treino,
**so that** o sistema aprenda minhas preferências e melhore os próximos.

## Acceptance Criteria
1. Tela de avaliação com rating 1–5 estrelas
2. Opções rápidas de feedback: "Muito fácil" / "Na medida" / "Muito difícil" / "Não gostei"
3. Campo opcional de comentário
4. Avaliação salva em `workout_evaluations`
5. Tela de celebração com animação após salvar
6. Botão para voltar ao dashboard
