# Story 4.2: Sistema de Conquistas

## Status
Done

## Story
**As a** usuário engajado,
**I want** desbloquear conquistas ao atingir marcos na minha jornada de treinos,
**so that** me sinta recompensado pela consistência e progresso.

## Acceptance Criteria
1. Após salvar avaliação, sistema verifica conquistas não desbloqueadas
2. Critérios verificados: treinos_totais, streak, level_up, horario, avaliacoes
3. Novas conquistas inseridas em user_achievements (idempotente via unique constraint)
4. Tela de achievement unlock com animação e emoji/nome da conquista
5. Fluxo: Avaliação → (Level Up?) → (Achievements?) → Celebração → Dashboard
