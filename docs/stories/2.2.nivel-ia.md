## Story
**As a** usuário que completou a anamnese,
**I want** que o sistema defina meu nível de treino automaticamente com base nas minhas respostas,
**so that** meus treinos sejam adequados ao meu condicionamento físico atual desde o primeiro dia.

## Acceptance Criteria
1. Lógica de classificação aplicada automaticamente após conclusão do quiz com as seguintes regras:
   - **Adaptação**: sedentário (`nunca_treinou` ou `mais_1_ano`) OU doença cardíaca OU lesão crônica ativa
   - **Iniciante**: `6m_1ano` sem treinar, sem condições de risco
   - **Intermediário**: `menos_6_meses` sem treinar (treinou até recentemente)
   - **Avançado** e **PRO**: definidos manualmente pelo usuário em futuras atualizações (no MVP, máximo Intermediário para novos usuários)
2. Tela de revelação do nível exibida com: nome do nível, descrição motivacional, ícone/ilustração
3. Animação de revelação (fade + scale) ao mostrar o nível
4. Exibição de quantos treinos são necessários para evoluir ao próximo nível
5. Nível salvo no campo `nivel_atual` em `user_profiles`
6. Botão "Começar" que avança para a próxima etapa do onboarding (`/onboarding/local`)

## Dev Notes

### Lógica de Classificação
```typescript
// src/lib/training/level-classifier.ts
export type TrainingLevel = 'adaptacao' | 'iniciante' | 'intermediario' | 'avancado' | 'pro'

export interface QuizAnswers {
  tempo_sem_treinar: 'menos_6_meses' | '6m_1ano' | 'mais_1_ano' | 'nunca_treinou'
  doenca_cardiaca: boolean
  lesao_cronica: boolean
}

export function classifyLevel(answers: QuizAnswers): TrainingLevel {
  // Regra de segurança tem prioridade máxima
  if (answers.doenca_cardiaca || answers.lesao_cronica ||
      answers.tempo_sem_treinar === 'nunca_treinou' ||
      answers.tempo_sem_treinar === 'mais_1_ano') {
    return 'adaptacao'
  }
  if (answers.tempo_sem_treinar === '6m_1ano') return 'iniciante'
  if (answers.tempo_sem_treinar === 'menos_6_meses') return 'intermediario'
  return 'adaptacao' // fallback seguro
}
```
