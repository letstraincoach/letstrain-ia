# Story 2.4: Mapeamento de Equipamentos (Academia de Condomínio)

## Status
Done

## Story
**As a** usuário de academia de condomínio,
**I want** fotografar os aparelhos disponíveis e ter a IA identificá-los,
**so that** meus treinos usem apenas o que está na minha academia.

## Acceptance Criteria
1. Tela de upload com instrução clara e suporte a múltiplos arquivos (até 10 fotos)
2. Fotos enviadas para Supabase Storage (`equipment-photos/{user_id}/`)
3. Claude API (vision) analisa as fotos e retorna lista de equipamentos detectados
4. Usuário vê a lista detectada e pode confirmar, remover ou adicionar manualmente
5. Lista final salva em `user_equipment`
6. Fallback: se IA não detectar nada, mostrar lista manual para seleção
