## Story
**As a** usuário,
**I want** me cadastrar e fazer login com email/senha ou conta Google,
**so that** eu possa acessar o app com segurança e minha sessão seja mantida entre visitas.

## Acceptance Criteria
1. Tela de cadastro com campos: nome, email, senha e confirmação de senha
2. Tela de login com email/senha e botão "Entrar com Google"
3. Google OAuth configurado via Supabase Auth e funcionando em produção
4. Fluxo de recuperação de senha via email funcionando (usuário recebe email, redefine senha)
5. Sessão persistida — usuário não precisa logar toda vez que abre o app
6. Após primeiro cadastro: redirecionar para `/onboarding/quiz`
7. Após login de usuário já cadastrado com onboarding completo: redirecionar para `/dashboard`
8. Após login de usuário com onboarding incompleto: redirecionar para onde parou no onboarding
9. Validação de formulários com mensagens de erro claras (campo obrigatório, email inválido, senhas não coincidem, senha muito curta)
10. Middleware de autenticação protegendo rotas do grupo `(app)` — redireciona para `/login` se não autenticado

## Dev Notes

### Middleware Pattern (Next.js 15 + Supabase SSR)
```typescript
// src/middleware.ts
import { createServerClient } from '@supabase/ssr'
import { NextResponse, type NextRequest } from 'next/server'

export async function middleware(request: NextRequest) {
  let supabaseResponse = NextResponse.next({ request })
  const supabase = createServerClient(
    process.env.NEXT_PUBLIC_SUPABASE_URL!,
    process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
    {
      cookies: {
        getAll() { return request.cookies.getAll() },
        setAll(cookiesToSet) {
          cookiesToSet.forEach(({ name, value }) => request.cookies.set(name, value))
          supabaseResponse = NextResponse.next({ request })
          cookiesToSet.forEach(({ name, value, options }) =>
            supabaseResponse.cookies.set(name, value, options)
          )
        },
      },
    }
  )
  const { data: { user } } = await supabase.auth.getUser()
  const isAuthPage = request.nextUrl.pathname.startsWith('/login') || request.nextUrl.pathname.startsWith('/cadastro')
  const isAppPage = request.nextUrl.pathname.startsWith('/dashboard') || request.nextUrl.pathname.startsWith('/workout') || request.nextUrl.pathname.startsWith('/progress') || request.nextUrl.pathname.startsWith('/settings')

  if (!user && isAppPage) {
    return NextResponse.redirect(new URL('/login', request.url))
  }
  if (user && isAuthPage) {
    return NextResponse.redirect(new URL('/dashboard', request.url))
  }
  return supabaseResponse
}

export const config = {
  matcher: ['/((?!_next/static|_next/image|favicon.ico|api/health).*)'],
}
```

### Google OAuth Callback
Supabase cuida do callback automaticamente via `/auth/v1/callback`. Basta configurar no Google Cloud Console e no Supabase Dashboard. No Next.js, criar `src/app/auth/callback/route.ts`:
```typescript
import { createClient } from '@/lib/supabase/server'
import { NextResponse } from 'next/server'
export async function GET(request: Request) {
  const { searchParams, origin } = new URL(request.url)
  const code = searchParams.get('code')
  if (code) {
    const supabase = await createClient()
    await supabase.auth.exchangeCodeForSession(code)
  }
  return NextResponse.redirect(`${origin}/dashboard`)
}
```

### Validação com Zod
```typescript
// src/lib/validations/auth.ts
import { z } from 'zod'
export const cadastroSchema = z.object({
  nome: z.string().min(2, 'Nome muito curto'),
  email: z.string().email('Email inválido'),
  senha: z.string().min(8, 'Mínimo 8 caracteres'),
  confirmarSenha: z.string(),
}).refine((d) => d.senha === d.confirmarSenha, {
  message: 'Senhas não coincidem', path: ['confirmarSenha'],
})
```
