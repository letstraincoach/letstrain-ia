# Story 6.1: Configuração de Push Notifications

## Status
Done

## Story
**As a** usuário,
**I want** ativar notificações e configurar meus dias e horários de treino,
**so that** o app me lembre de treinar automaticamente.

## Acceptance Criteria
1. Service Worker registrado (public/sw.js) para receber push em background
2. Componente PushNotificationSetup solicita permissão via Web Push API
3. Após permissão: usuário escolhe dias da semana + horário de lembrete
4. Configuração salva em push_subscriptions (endpoint + keys VAPID + dias + horário)
5. Disponível na tela /settings junto com dados do perfil e assinatura
