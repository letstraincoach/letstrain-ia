# Story 3.2: Tela de Treino do Dia

## Status
Done

## Story
**As a** usuário com treino gerado,
**I want** ver e executar meu treino em uma tela clara e intuitiva,
**so that** eu possa completar todos os exercícios com as informações certas na palma da mão.

## Acceptance Criteria
1. Header com nome do treino, duração estimada e nível
2. Navegação entre exercícios: anterior / atual / próximo com indicador de progresso
3. Card do exercício atual com: nome, grupo muscular, séries × repetições, descanso, instruções
4. Botão "Ver Vídeo" abrindo YouTube em nova aba
5. Botão "Concluir Treino" ao finalizar todos os exercícios
6. Modal de confirmação antes de concluir
7. Ao confirmar: chamar API para marcar treino como executado → redirecionar para avaliação
