# Story 5.1: Planos e Checkout

## Status
Done

## Story
**As a** usuário sem assinatura,
**I want** ver os planos disponíveis e assinar via cartão ou PIX,
**so that** eu possa acessar todos os treinos do app.

## Acceptance Criteria
1. Tela /assinatura com cards de plano: Mensal (R$29,90) e Anual (R$249,00 + destaque de economia)
2. Botão "Assinar" cria preferência no Mercado Pago e redireciona para Checkout Pro
3. MP Checkout suporta cartão de crédito/débito e PIX nativamente
4. Webhook POST /api/webhooks/mercadopago processa payment.approved
5. Assinatura criada na tabela subscriptions via service role (bypass RLS)
6. Redirecionamento para /assinatura/sucesso (cartão) ou /assinatura/pendente (PIX)
