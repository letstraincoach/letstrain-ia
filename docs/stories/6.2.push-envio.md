# Story 6.2: Envio de Push Notifications

## Status
Done

## Story
**As a** sistema,
**I want** enviar push notifications nos horários configurados e ao desbloquear conquistas,
**so that** o usuário seja lembrado e celebrado automaticamente.

## Acceptance Criteria
1. Utility sendPushToUser() envia push via web-push (VAPID) para todos os devices do usuário
2. Supabase Edge Function (send-push-reminders) executa cron a cada hora, filtra subscriptions ativas pelo dia+hora configurados
3. Edge Function verifica assinatura ativa antes de enviar (não notifica usuários vencidos)
4. post-complete route dispara push imediato ao fazer level-up
5. post-complete route dispara push ao desbloquear novas conquistas
6. Notificação de lembrete inclui mensagem motivacional personalizada
7. Notificação de achievement inclui emoji e nome da conquista
